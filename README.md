# nanhang-frontend

## 启动方式

本项目是纯静态前端页面，推荐使用本地静态服务器启动。

```bash
cd /Users/hh/MacCodes/nanhang-frontend
python3 -m http.server 5500
```

浏览器访问：

- `http://localhost:5500/index.html`

停止服务：

- 在终端按 `Ctrl + C`
